Introducción a NodeJS

¿Qué es NodeJS?

Tal como podemos encontrar en su página oficial

Node.js® es un entorno de ejecución para JavaScript construido con el motor de JavaScript V8 de Chrome. Node.js usa un modelo de operaciones E/S sin bloqueo y orientado a eventos, que lo hace liviano y eficiente. El ecosistema de paquetes de Node.js, npm, es el ecosistema mas grande de librerías de código abierto en el mundo.

Node.js usa un modelo de operaciones E/S asíncrono y orientado a eventos, que lo hace liviano y eficiente.

Un modelo asíncrono es un modelo donde las operaciones no se realizan
con un orden determinado.

En conclusión...

La programación dirigida por eventos es un paradigma de programación en el que el flujo del programa está determinado por eventos o mensajes desde otros programas o hilos de ejecución.
