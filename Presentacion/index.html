Node.js® es un entorno de ejecución para JavaScript construido con el motor de JavaScript V8 de Chrome.

  • V8 es el motor de código abierto JavaScript
    de alto rendimiento de Google
  • Escrito en C ++
  • Utilizado en Google Chrome

Node.js usa un modelo de operaciones E/S asíncrono y orientado a eventos, que lo hace liviano y eficiente.

(Cuando nos referimos al modelo de operaciones, nos referimos a cómo maneja la concurrencia)

Un modelo asíncrono es un modelo donde las operaciones no se realizan
con un orden determinado.

En JavaScript, usamos Callbacks o promesas para manejar esta asincronía

En conclusión...

La programación dirigida por eventos es un paradigma de programación en el que el flujo del programa está determinado por eventos o mensajes desde otros programas o hilos de ejecución.
